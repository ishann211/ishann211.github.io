Project Overview

This project is a complete IoT-based smart agriculture monitoring system designed to enhance farming decisions by collecting, analyzing, and visualizing real-time environmental and soil data. The list of parameters are,

  • NPK (Nitrogen, Phosphorus, Potassium) values of soil
  • Soil Moisture
  • Rain gauge
  • Sunlight Intensity
  • Temperature
  • Humidity

The system uses multiple hardware devices to capture sensor readings and transmits them via MQTT to Firestore, enabling live monitoring through a mobile app (Flutter) and web app (React + Vite).

An independent ML model (YOLOv5) processes aerial images to detect tree count and assess tree health, hosted via Cloud Run.

Hardware Devices,

  • Portable Unit- Field workers collect soil data using ESP32, NPK, Moisture, GPS, RTC, SD card, LCD. Stores data offline when Wi-Fi is unavailable.
  • Fixed Unit- Continuously monitors field data. Sends periodic readings to the cloud via MQTT with Wi-Fi and status indicators.

Mobile App,

Built with Flutter + Firebase, provides real-time and historical sensor data, interactive graphs, offline sync, and alert systems.

Web App,

Made with React + Vite, hosted on Firebase, offering live dashboards, trend analysis, geo-mapping of devices, and sensor status.

ML Model,

Uses YOLOv5 to analyze aerial images and detect tree count and tree health (Healthy/Unhealthy). Deployed on Google Cloud Run, accessible via API for web/mobile input.

Outcome,

Accurate, geo-tagged insights for precision farming with a deployment-ready architecture.

Key Features

Real-time IoT data over MQTT

Continuous real-time environmental data transmission via lightweight MQTT messaging protocol architecture

Offline logging with RTC+SD

Reliable offline sensor data storage using RTC-based timestamping and SD persistence

AI-powered plant health analysis

Automated plant health assessment using deep learning-based image classification techniques

Scalable Firestore backend

Cloud-native Firestore backend enabling scalable, structured, and secure data management

Mobile + Web visualization

Cross-platform mobile and web dashboards for interactive environmental data visualization
